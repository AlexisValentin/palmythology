# Notes de version 2.17.5 (27/01/2026)

## Les Grandes Lignes

- Correction d'un bug qui faisait que les images du carousel laissaient un espace blanc sur des zoom réduit du navigateur web.

## Quoi 2 Neuf

- Correction d'un bug qui faisait que le nouveau texte d'aperçu ne s'affichait pas sur écrans mobiles lorsque le lien est accessible.
- Ajustement de marges CSS pour les écrans bureaux et mobiles. 

## Godle

- Correction d'un bug qui faisait que les résultats ne se réinitialisaient pas à minuit heure de Paris, malgré la réinitialisation de l'entité mythologique.

# Divers

- Optimisation du chargement des images:
  - Utilisation du composant `next/image`.
  - Intégration de props `sizes` pour définir une valeur fixe pour chaque image.
  - Arrêt de l'optimisation des images vectorielles.
